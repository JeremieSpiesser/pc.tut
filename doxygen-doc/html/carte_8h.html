◆ getEffet()

char * getEffet ( carte  c)

donne un char* sécrivant les effets de la carte

Paramètres
c
Renvoie
retourne un char* décrivant les effets de la carte. Il est obtenu par concaténation de deux mots, l'un pour les effets spécifiques aux cartes personnel, l'autre pour les cartes actions Exemple: pour la carte 21 (carte action) : "00000000000010000000000\0" pour la carte 9 (Dimitri Watel, carte personnel) : "22000000000100000000000\0" Comme pour le char* des cartes personnel, on numérote les effets des cartes actions Pour une carte élève on renvoie NULL
